# 🌸 Iris Classification using K-Nearest Neighbors (KNN)

This project implements the **K-Nearest Neighbors (KNN)** algorithm to classify iris flowers into different species based on their features.

---

## 📌 Project Overview

The goal of this project is to classify iris flowers into three species:

* Setosa
* Versicolor
* Virginica

The classification is based on:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

---

## 📊 Dataset

The project uses the famous **Iris dataset**, which contains 150 samples of iris flowers with 4 features each.

* Algorithm: K-Nearest Neighbors (KNN)
